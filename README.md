# ssl-ayx-guide

## 项目简介

`ssl-ayx-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为一组静态 HTML 文件的集中存放与展示空间。

这些页面内容可能涵盖技术说明、参考指南、个人笔记或实用工具等，以纯前端形式呈现，便于直接访问和阅读。

## 目录说明

```
ssl-ayx-guide/
├── index.html          # 仓库主页 / 导航页
├── pages/              # 存放各独立 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
├── README.md           # 本文件
└── LICENSE             # 许可证文件
```

- `index.html`：仓库的入口页面，通常包含页面列表或目录。
- `pages/`：所有独立 HTML 页面归档于此。
- `assets/`：页面所需的样式、脚本、图片等资源文件。

## 页面归档说明

本仓库中的每个 HTML 页面均为独立完整的前端文档，不依赖后端服务。页面之间没有强关联，可根据需要单独访问。

页面内容由仓库维护者定期整理和更新，归档时尽量保持结构清晰、资源自包含。

如需添加新页面，请按照以下步骤操作：

1. 在 `pages/` 目录下创建新的 HTML 文件。
2. 如有需要，将相关资源放入 `assets/` 对应子目录。
3. 更新 `index.html` 中的导航链接（如适用）。
